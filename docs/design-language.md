# Design Language — Capture PWA

> 방향: **토스(Toss)-derived** — 군더더기 없는 자신감 + 친근함. 어두운 무드가 아니라 *밝고 깨끗하고 명료*. (사용자 결정, 2026-06-29)
> 단 하나의 예외: **카메라 뷰파인더는 유일한 다크 풀스크린 순간** — 라이브 영상 자체가 색이므로.

## 정체성 한 줄
"읽기를 멈추지 않게 하는, 가볍고 확신에 찬 캡처 도구." 토스가 '송금'에서 마찰을 없앴듯, Capture는 '생각 붙잡기'에서 마찰을 없앤다.

---

## Palette (토스 팔레트 기반, 의도적 선택)

| 역할 | Hex | 용도 |
|------|-----|------|
| Ink (주 텍스트) | `#191F28` | 제목·본문. 순흑이 아닌 토스 잉크 |
| Sub (보조 텍스트) | `#8B95A1` | 캡션·메타(시각·저자) |
| Surface | `#F2F4F6` | 카드·입력칸·칩 배경 |
| Background | `#FFFFFF` / `#F9FAFB` | 앱 기본 바탕 |
| **Primary (브랜드)** | `#3182F6` | CTA·선택·강조. **단 하나의 파랑** |
| Viewfinder (다크) | `#17171C` | 카메라 화면 전용 풀블리드 다크 |

- **색 규율:** UI는 무채(잉크/그레이/화이트) + 파랑 하나. **컬러는 ① 라이브 카메라 ② 태그(💡⭐🔗❓🌱)에서만** 터진다. 태그가 화면의 색 이벤트.

## Typography — Pretendard 단일 슈퍼패밀리

토스처럼 *한 가족*으로 위계를 만든다(여러 폰트 짜깁기 안 함 = 의도적 절제).
- **Display/제목:** Pretendard **Bold~ExtraBold**, 큼직하게(28–34px), 타이트한 자간. "지금 읽는 책" 같은 헤드라인이 화면을 연다.
- **Body/UI:** Pretendard Regular/Medium (16–17px).
- **Data/숫자:** Pretendard + **tabular-nums** (캡처 수·시각). 별도 모노 없이 정렬만 또렷하게.
- "왜 저장했나요?"는 Display 중간 굵기로 — 폼 라벨이 아니라 *질문*처럼.

## Shape & Space
- **Radius:** 카드 16px · 버튼 14px · 칩/태그 full pill.
- **여백:** 넉넉하게. 좌우 거터 20–24px, 요소 간 16–20px. 비어 있음을 두려워하지 않는다.
- **CTA:** 하단 고정 **풀폭 솔리드 파랑 버튼**, 높이 ~56px, 볼드 화이트. ("▶ 독서 시작", "저장")
- 한 손 도달: 모든 주요 조작은 화면 하단 thumb-zone.

## Motion — 빠르고 탄력 있게, 단 예산 안에서
- 토스의 시그니처는 *스프링감*. `cubic-bezier` 오버슛으로 탱탱하게.
- **단, 캡처 루프 모션은 < 100ms** (16장 예산). 지연을 만드는 장식 모션 금지.
- `prefers-reduced-motion` 존중.

## Signature — "저장의 쾌감"
이 앱이 기억될 단 하나의 순간 = **캡처 저장 피드백.** 토스의 '완료' 모먼트처럼:
- 셔터 → 프레임이 살짝 굳고 → 태그가 스프링으로 **'팝'** 하며 선택됨(잉크 스탬프 느낌, 태그색).
- 저장 시 짧고 만족스러운 확인(체크 + 햅틱) → 즉시 카메라 복귀.
- 캡처 목록 카드엔 그 태그색 pill이 정체성으로 남음.
- *과장 금지* — 한 번의 작은 쾌감, 그 외엔 조용히.

## 화면별 적용
- **Home / Review / Export / 목록:** 밝은 토스-클린. 흰 바탕, 큰 볼드 헤드라인, 둥근 카드 리스트, 하단 파랑 CTA.
- **Capture(카메라):** 유일한 다크 풀블리드. 상단 반투명 pill(📚책·🎯프로젝트·📍수), 하단 클린 패널에 큰 태그 버튼 + 셔터. 셔터 후 "왜 저장했나요?" 라운드 바텀시트가 스프링으로 올라옴(칩 + 큰 저장 버튼).

## Voice (마이크로카피 — 토스식)
- plain·sentence case·사람 말투. "제출"❌ → "저장"✅. 액션은 끝까지 같은 이름.
- 빈 화면은 초대장: "첫 생각을 붙잡아 보세요." 에러는 사과하지 않고 다음 행동을 안내.

## 품질 바닥선
모바일 반응형 · 키보드 포커스 가시 · reduced-motion 준수 · 큰 탭 타깃(≥48px).
